ETL Process Overview

This diagram illustrates the ETL (Extract, Transform, Load) pipeline used in the project to collect, clean, and analyze data from the targeted source.

1. First of all, data is extracted from the armed.forcedeu website using Python scripts. And also with SIPRI data for enrichment.

2. Then, the raw data undergoes a transformation process that includes cleaning, type conversion, and structuring to ensure consistency and usability.

3. Finally, the transformed data is loaded into a MySQL database for efficient storage and retrieval. Power BI is then used to create visualizations and reports based on the cleaned data.

The two reports linked below showcase the extraction of data from the armed.forcedeu website, using Python and first data analysis.

Then SQL is used to create databases and tables to store the cleaned data and make queries. Finally, Power BI is used to create visualizations and reports based on the cleaned data.

Mémoire de Master : Opportunités d'emploi salarié et création d'entreprises

Résumé

Ce mémoire explore les déterminants de la création d'entreprise en se concentrant sur la motivation des entrepreneurs liée à leurs opportunités d'emploi. En retraçant la littérature sur les entrepreneurs motivés par opportunité et par nécessité, nous avons vu que ces deux types d’entrepreneurs sont identifiés par leur statut antérieur à la phase de création (Fairlie & Fossen, 2018).

Notre étude remet en question cette identification des entrepreneurs d'opportunité et de nécessité, soulignant l'importance d'une approche nuancée basée sur les options d'emploi réelles. Nous travaillons sur les données de l'Enquête Emploi en continu (INSEE) en France.

Nous avons utilisé la méthodologie de l’Outside Option Index (OOI) développée par Caldwell et Danieli (2020) pour mesurer les opportunités extérieures des individus sur le marché du travail. Nos résultats indiquent que les chômeurs avec une Outside Option plus élevée ont une probabilité légèrement plus élevée de se lancer dans l’entreprenariat.

En revanche, une Outside Option élevée pour les salariés semble réduire leur volonté à devenir entrepreneurs et créer leur entreprise.

Comparative Analysis of Machine Learning Models for Water Potability Prediction

A comprehensive machine learning project that compares multiple algorithms to predict water potability, helping ensure safe drinking water access.

Machine Learning Approach

This project implements and compares various machine learning models including Random Forest, SVM, Logistic Regression, and Neural Networks to predict water safety.

The analysis includes feature engineering, model validation, cross-validation techniques, and performance metrics comparison to identify the most effective prediction approach.

Results demonstrate the effectiveness of ensemble methods in achieving high accuracy for water quality classification, with practical implications for water safety monitoring.
